Pagination Usage

Pagination bileşeninin kullanımı.

{% raw %}{% set pagination_data = {
    "current_page": 3,
    "total_pages": 10,
    "pages": [
        {"number": 1, "url": "?page=1", "active": false},
        {"number": 2, "url": "?page=2", "active": false},
        {"number": 3, "url": "?page=3", "active": true},
        {"number": 4, "url": "?page=4", "active": false},
        {"number": 5, "url": "?page=5", "active": false}
    ],
    "has_previous": true,
    "has_next": true,
    "previous_page": 2,
    "next_page": 4,
    "size": "", // Optional: sm, lg
    "alignment": "", // Optional: center, end
    "rounded": false, // Optional: true for rounded style
    "show_first_last": false // Optional: true to show first/last buttons
} %}

{% include "components/_pagination.html" with pagination_data %}{% endraw %}

Alternatif olarak, UI Service'i kullanarak pagination verisi oluşturabilirsiniz:

{% raw %}from core.services.ui_service import get_ui_service

# UI Service'den pagination verisi oluştur
ui_service = get_ui_service()
pagination_data = ui_service.create_pagination(
    current_page=3,
    total_pages=10,
    url_pattern="?page={page}"
)

# Template'e aktar
return render_template('my_page.html', pagination=pagination_data){% endraw %}
